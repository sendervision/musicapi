## 🔌 Running Locally

> [!NOTE]
> You need `Bun(1.0.26+)` or `Node.js(v18+)`

1. Clone the repository:

   ```sh
   git clone https://github.com/sendervision/musicapi
   cd musicapi
   ```

2. Install the required dependencies:

   ```sh
   bun install
   ```
   or
   ```sh
   npm install
   ```


3. Launch the development server:

   ```sh
   bun run dev
   ```
   ```sh
   npm run dev
   ```
